## T0.5. The Two-Stage Observability Filter ✅

**Theorem T0.5 (Spectral acceptance window).** A mode $(n,d)$ belongs to $\sigma_{\rm phys}(|D|)$ if and only if it passes both stages of the acceptance window on the projected resolvent of $D$:

**Stage 1 — Projection amplitude (gain window).** The mode must have sufficient overlap with the 4D slice. The mismatch is

$$\Omega_{\log}(n,d) = \ln\frac{S(n,d)}{S(n,2)}.$$

A mode passes Stage 1 iff $\Omega_{\log} \leq \ln 2$. Modes exceeding this threshold are exponentially suppressed in the projected amplitude — exactly the laser-cavity gain window condition.

**Stage 2 — Co-fixed-point stability (longitudinal mode condition).** The mode index $n$ must be a fixed point or beat-frequency of the sector comb filtration generated by $n_s$. Modes that are not co-fixed points decohere on the timescale $1/m_{\mathrm{scale},d}$ and do not appear as stable resonances — exactly the longitudinal mode condition in a cavity.

Together Stage 1 selects the sectors (gain bandwidth) and Stage 2 selects the stable resonances within each sector. The result is the 15-element set $\sigma_{\rm phys}(|D|)$.

$$\boxed{m(n,d) = S(n,d)\times m_{\mathrm{scale},d}, \qquad S(n,d) = \binom{n+d-1}{d}}$$

$$P_d(t) = \frac{1}{(1-t)^d} = \sum_{n\geq 1} S(n,d)\,t^n.$$

**Proof.** $S(n,d)=\binom{n+d-1}{d}$ counts the degree-$(n-1)$ monomials in $d$ variables, equals $\dim\mathrm{Sym}^{n-1}(\mathbb{R}^{d+1})$, and equals the IDOS (integrated density of states) of the $d$-dimensional harmonic oscillator at level $n$ — the number of eigenstates with quantum number $\leq n-1$. In laser cavity terms: the cumulative count of transverse modes up to order $n-1$ in a $(d-1)$-transverse-mode cavity. $\square$

$$\boxed{S(n,d) \;=\; \dim\mathrm{Sym}^{n-1}(\mathbb{R}^{d+1}) \;=\; \mathrm{IDOS}_d(n) \;=\; [t^n]\frac{1}{(1-t)^d}}$$
